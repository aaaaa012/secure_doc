# Glossary

| Term | Definition |
|------|------------|
| **API** | Application Programming Interface. A set of rules that allows different software entities to communicate with each other. |
| **JWT** | JSON Web Token. A compact, URL-safe means of representing claims to be transferred between two parties. Used here for secure authentication. |
| **QR Code** | Quick Response Code. A machine-readable optical label that contains information about the item to which it is attached. |
| **Endpoint** | A specific URL where an API can be accessed by a client application. |
| **MVP** | Minimum Viable Product. A version of a product with just enough features to be usable by early customers. |
| **TLS** | Transport Layer Security. A cryptographic protocol designed to provide communications security over a computer network. |
| **CRUD** | Create, Read, Update, Delete. The four basic functions of persistent storage. |
